\section{Supplementary: Compute Ledger and Gate Structure}

The JSONL ledger (\texttt{research/ledger.jsonl}) records every experiment proposal, disposition, spend,
and amendment. Each gate JSON (\texttt{gates/gate\_L*.json}) follows the schema:

\begin{lstlisting}[language=json, caption=Gate JSON Schema]
{
  "gate_id": "L9_alignconf",
  "tier": "confirm",
  "pre_registration": {...},
  "amendments": [...],
  "domain_gate": {
    "verdict": "PASS",
    "margin_nats": 0.12,
    "evidence": "{...}"  // JSON string with per-seed results
  },
  "cost": {"compute_hours": 2.3, "gpu_model": "H100"},
  "review_history": [{"review_id": 9, "verdict": "PASS"}]
}
\end{lstlisting}

All gates, amendments, and reviews are auditable in the GitHub repository.
